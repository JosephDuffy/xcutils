  • archiveApp(at:outputTo:enableVerboseLogging:)

    Builds a zip archive containing the app at the provided URL.

    This uses the ditto command line tool to create an archive similar to Finder’s “Compress MyApp.app”, which is suitable for upload to Apple’s servers, notarization, etc.
